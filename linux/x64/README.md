# SimpleOS Linux x64 System

Bu dizin, SimpleOS için bir x64 Linux sistem demosu içerir.

## Derleme

```bash
cd linux/x64
make
```

## Çalıştırma

```bash
./x64_system --help
./x64_system --info-only
./x64_system --demo-only
```

## Açıklama

- `x64_system` 64-bit Linux x64 hedefi için hazırlanmış bir demo programdır.
- Komut satırı seçenekleri mimari bilgisi ve demo çalıştırma davranışını ayrıştırır.
