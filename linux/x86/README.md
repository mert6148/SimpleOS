# SimpleOS Linux x86 System

Bu dizin, SimpleOS için bir x86 Linux sistem demosu içerir.

## Derleme

```bash
cd linux/x86
make
```

## Çalıştırma

```bash
./x86_system --help
./x86_system --info-only
./x86_system --demo-only
```

## Açıklama

- `x86_system` 32-bit Linux x86 hedefi için hazırlanmış bir demo programdır.
- Komut satırı seçenekleri mimari bilgisi ve demo çalıştırma davranışını ayrıştırır.
