# Google Documentation Java SDK

This folder contains a simple Java SDK project for Google Documentation workflows.

## Project structure

- `pom.xml` - Maven project descriptor
- `src/main/java/com/googledocumentation/sdk/` - Java SDK source code

## Build and run

```powershell
cd "google documantation/src"
mvn package
java -cp target/google-documentation-java-sdk-0.1.0.jar com.googledocumentation.sdk.App
```

## Features

- Document modeling
- Document management service
- Search and HTML preview
