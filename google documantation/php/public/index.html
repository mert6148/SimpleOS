Başlarken

Bu modül, dokümantasyon içeriğini Markdown dosyalarından yükleyip statik HTML çıktısı olarak üretir.

Özellikler

  • Markdown dosyalarını tarar
  • Başlıkları ve listeleri işler
  • Basit HTML çıktısı üretir
  • bin/generate-docs.php ile dokümanları public/index.html olarak oluşturur

Çalıştırma

```bash

php bin/generate-docs.php

```
