1. Temel Tablo

En basit tablo kullanımı. Veri ve sütunları tanımlayıp oluşturuyoruz.

const table = new Table("#table-basic", columns, data, { caption: "Temel Tablo Örneği", pageSize: 5 });

2. Gelişmiş Özellikler

Sıralama, sayfalama, seçim ve arama özelliklerini içeren tablo.

const table = new Table("#table-advanced", columns, data, { sortable: true, selectable: true, searchable: true, pageSize: 10, caption: "Gelişmiş Tablo" });

3. Koyu Tema

Koyu tema desteği ile göz yormayan tasarım.

const table = new Table("#table-dark", columns, data, { theme: "dark", pageSize: 5 });

4. CSV/JSON Dışa Aktarma

Tablo verilerini CSV veya JSON formatında indirebilirsiniz.

const table = new Table("#table-export", columns, data, { exportable: true, pageSize: 5, caption: "Dışa Aktarılabilir Tablo" });

5. Sütun Gizleme/Gösterme

Kullanıcılar sütunları seçerek gösterebilir veya gizleyebilir.

const table = new Table("#table-filter", columns, data, { columnHideable: true, pageSize: 5, caption: "Filtrelenebilir Sütunlar" });
